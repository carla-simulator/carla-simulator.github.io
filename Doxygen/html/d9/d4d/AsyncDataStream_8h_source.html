23 /// A streaming channel for sending sensor data to clients, supports sending
24 /// data asynchronously. Data sent by the "Send" functions is passed to the
25 /// serializer registered with the sensor at carla::sensor:SensorRegistry before
26 /// being sent down the stream.
27 ///
28 /// @warning This is a single-use object, a new one needs to be created for each
29 /// new message.
30 ///
31 /// FAsyncDataStream also has a pool of carla::Buffer that allows reusing the
32 /// allocated memory, use it whenever possible.
